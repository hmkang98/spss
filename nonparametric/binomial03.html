이항 검정이 올바르게 실행할 수 있도록 여러 단계를 설정해보자.
  • Test Variable List는 검정에 사용할 변수를 입력하고
  • Test Proportion
  • Define Dichotomy는 자료를 두 집단으로 구분하는 메뉴로
    • Get from data는 두 집단으로 나눌 변수가 있는 경우에 사용하고
    • Cut point는 부호를 나눌 기준이 되는 숫자를 직접 입력하여
    변수를 두 집단으로 나눈다.
위의 단계가 설정되었으면 OK 버튼을 클릭하여 결과를 얻는다.
