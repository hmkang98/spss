Mann_Whitney test가 올바르게 실행할 수 있도록 여러 단계를 설정해보자.
  • Test Variable List는 검정에 사용할 변수를 입력하고
  • Test Type은 검정 방법을 선택하는 메뉴로 Mann-Whitney U를 선택하며
  • Group Variable은 그룹을 나누는 메뉴로 교육방법 변수를 추가한 다음
  • Define Range 버튼을 클릭하면 팝업 창이 나타나며 이 창에 Group Variable에 입력된 자료의 범위를 입력한다.
위의 단계가 설정되었으면 OK 버튼을 클릭하여 결과를 얻는다.
