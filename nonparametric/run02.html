런 검정이 올바르게 실행할 수 있도록 여러 단계를 설정해보자.
  • Test Variable List는 검정에 사용할 변수를 입력하고
  • Test Proportion
  • Cut Point는 자료를 두 집단으로 구분하는 메뉴로
    • Median은 중앙값
    • Mean은 평균
    • Mode는 최빈값
    • Custom 사용자가 구분값을 직접 설정하여
    변수를 두 집단으로 나눈다.
위의 단계가 설정되었으면 OK 버튼을 클릭하여 결과를 얻는다.
