막대 그래프(bar chart)

사용자가 그릴 막대 도표를 선택한다.
  • 단순 : 한 변수에 대하여 막대그래프를 작성하는 경우
  • 수평누적 : 막대그래프를 작성할 변수를 설정하고 다시 수평으로 구분할 변수를 설정하여 그래프를 작성
  • 수직누적 : 막대그래프를 작성할 변수를 설정하고 다시 수직으로 구분할 변수를 설정하여 그래프를 작성
여기에서는 "수평누적" 형태의 막대 도표를 그린 경우를 보여준 것이다. "도표에 표시할 데이터"는
  • "케이스 집단들의 요약값" : 각 범주별로 빈도수를 계산하여 막대도표를 작성.
  • "개별 변수의 요약값" : 변수를 자료값으로 하여 막대 도표를 작성.
  • "각 케이스의 값" : 이미 계산된 자료값을 막대 도표로 작성.
에서 하나를 선택한다.
